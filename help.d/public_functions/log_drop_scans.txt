
Name: log_drop_scans

Description:
	Globally LOG and/or DROP (nmap) scans.
	The following configuration variables can be configured:
	`DROP_Stealth_XMAS'
	`DROP_Stealth_XMAS_PSH'
	`DROP_Stealth_XMAS_ALL'
	`DROP_Stealth_FIN'
	`DROP_Stealth_SYN_RST'
	`DROP_Stealth_SYN_FIN'
	`DROP_Stealth_Null'

	`LOG_Stealth_XMAS'
	`LOG_Stealth_XMAS_PSH'
	`LOG_Stealth_XMAS_ALL'
	`LOG_Stealth_FIN'
	`LOG_Stealth_SYN_RST'
	`LOG_Stealth_SYN_FIN'
	`LOG_Stealth_Null'

Usage:
	log_drop_scans
	
Example:
	log_drop_scans
